# Changelog

TRPG/Murder Mystery Translator의 모든 버전별 변경 사항을 기록합니다.

## [v0.75] - 2026-03-26
### ✨ 신규 기능
- **Gemini Context Caching 적용**: 시스템 프롬프트(번역 규칙 + 용어집)를 서버 캐시에 올려 반복 과금 절감. 캐시 토큰은 일반 입력의 약 25%로 과금. 캐시 미지원 모델은 자동으로 일반 모드로 폴백
- **캐시 자동 복구**: 번역 도중 캐시가 만료(2시간)되면 자동으로 재생성 최대 3회 시도. 재생성 실패 시 중단하고 재시작 안내 메시지 표시
- **토큰 사용량 로그**: 청크 번역마다 입력/캐시/출력 토큰 수를 로그에 표시 (`[토큰] 입력 xxx (캐시 xxx) / 출력 xxx`)
- **CCFolia 메모 번역**: 코코포리아 룸 셋팅 탭에서 `__data.json` 내 `memo` 필드를 자동 번역하는 기능 추가. 일본어 감지 후 중복 제거하여 효율적으로 처리

### 🛠 개선 사항
- **기본 모델 변경**: `gemini-3-flash-preview` → `gemini-3.1-flash-lite-preview` (동일 품질, 약 50% 저렴)
- **테이블 기능 제거**: 오류 발생이 잦던 PDF 테이블 감지 및 번역 기능 제거
- **하이라이트 기능 제거**: 오류 유발 요인이던 `<hl=>` 하이라이트 태그 추출/번역 기능 제거
- **불필요한 프롬프트 규칙 제거**: 이미 삭제된 하이라이트 기능의 프롬프트 규칙(`<hl=>`) 제거로 토큰 절약
- **번역 실패 시 출력 파일 삭제**: 번역 오류로 중단된 파일의 불완전한 출력물을 자동 삭제하여 "이미 번역됨 건너뛰기" 기능이 오인하지 않도록 개선

### 🐛 버그 수정
- 시스템 프롬프트 규칙 번호 중복 문제 수정 (규칙 7번이 두 개 존재하던 문제)

## [v0.7] - 2026-03-16
### ✨ 신규 기능
- **코코포리아 룸 셋팅 탭 추가**: CoCoFolio 룸 ZIP의 이미지를 번역본으로 교체하고 `__data.json`의 SHA-256 해시 참조를 자동 업데이트
  - **수동 매핑 모드**: 번역본↔원본 이미지를 썸네일을 보며 직접 시각적으로 매핑
  - **파일명 자동 매칭 모드**: 동일 파일명끼리 자동 연결 (이미지 글자 제거 후 작업에 최적)
  - **API 자동 매칭**: Gemini Vision으로 번역본과 유사한 원본 이미지 자동 추천 (텍스트 내용 일치 우선 → 색상·구도 유사도 순)
  - 매핑 결과 JSON 저장/불러오기 지원 (반복 작업 시 재사용 가능)
  - ZIP 출력 옵션 (코코포리아 업로드용 `.zip` 자동 생성)

### 🛠 개선 사항
- 사용자 지정 아이콘 적용
- 수동 매핑 다이얼로그 썸네일 크기 개선 (120 → 180px)
- 원본 선택창 창 위치 기억 기능 추가
- API 자동 매칭 모델 별도 입력 지원 (기본값: `gemini-3.1-flash-lite-preview`)

## [v0.6] - 2026-03-10

- 하이라이트 태그(`<hl=...>`) 포함 구문의 색상 태그 패턴 인식 오류 수정

- API 호출 전 진행 로그 메시지 추가 (`N자 번역 중...`)

## [v0.5 beta] - 2026-02-26
### ✨ 신규 기능
- **이미지 글자 제거 (v0.5 beta)**: Gemini Pro Image 모델을 활용하여 이미지 내 텍스트를 배경 손지 없이 제거하는 기능 추가.
- **이미지 미리보기 다이얼로그**: 글자 제거 전/후를 실시간으로 비교할 수 있는 다이얼로그 추가.
- **알파 채널(투명도) 유지**: PNG 이미지의 투명 배경을 보존하는 옵션 추가.

---

## [v0.4] - 2026-02-24
### ✨ 신규 기능
- **DOCX 출력 지원**: 번역 결과를 워드 문서(`.docx`)로 자동 생성.
- **서식 복원**: DOCX 출력 시 원본의 글자 색상 및 폰트 스타일을 최대한 유지.
- **고유명사 자동 추출**: 시나리오 텍스트에서 인명/지명을 AI가 자동 분석하여 사전 파일 생성.

### 🛠 개선 사항
- **폴더 구조 유지**: 하위 폴더가 포함된 경로의 계층 구조를 출력 폴더에도 동일하게 재현.

## [v0.3] - 2026-02-08
### ✨ 신규 기능
- **PDF 이미지 선택**: PDF 내 이미지 중 번역이 필요한 항목만 체크박스로 선택하는 필터 기능 추가
